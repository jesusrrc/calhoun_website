# Root Folder

This folder contains the HTML, CSS, and Javascript files used to create the website. In order to open and explore the website locally, please open `Index.html`.

## `HTML` Folder

The `HTML` folder contains all the HTML files used to describe the webpages englobed in the website.

## `JS` Folder

The `JS` folder contains all the Javascript programs used in the *Converters* webpage.

## `CSS` Folder

The `CSS` folder contains the CSS file used to define the style used in the website's webpages.

## `Images` Folder

The `Images` folder contains the images and GIFs embedded throughout the website.
